Core Concepts in Theory of Computation

  • Formal languages: mathematical models of programming languages

  • Automata theory: abstract machines modeling computation

  • Computability: what problems can/cannot be solved algorithmically

  • Complexity theory: resource requirements for solving problems

Formal Languages

Employs mathematical tools to describe syntax and semantics of languages.

  • Alphabet: finite set of symbols (Σ)

  • String: finite sequence of symbols from alphabet

  • Language: set of strings following specific rules

  • Grammar: rules for generating valid strings

Automata Types

Automata are abstract machines that recognize languages. They also model computation.

  • Finite automata: DFA, NFA

  • Pushdown automata: stack memory

  • Linear-bounded automata: limited tape

  • Turing machines: infinite tape

Chomsky Hierarchy

Automaton works as recognizer for language. Types of automata are related to types of grammars.

  • Type-0: Unrestricted grammars → Turing machines

  • Type-1: Context-sensitive → Linear-bounded automata

  • Type-2: Context-free → Pushdown automata

  • Type-3: Regular → Finite automata
