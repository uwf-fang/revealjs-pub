  • Online ProLog Compiler

  • Local ProLog Compiler

    • Install SWI-Prolog

      • Linux sudo apt-get install swi-prolog

      • Mac OS brew install swi-prolog

    • Run SWI-Prolog swipl -s hello.pl

    $ sudo apt-get install swi-prolog
$ swipl -s hello.pl
