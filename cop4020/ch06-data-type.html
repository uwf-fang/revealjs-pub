Introduction to Data Types

  • Definition: Collection of data values and predefined operations

  • Read textbook chapter 6

  • Object-oriented perspective

    • Primitive data types only: C, FORTRAN, Pascal, ALGOL

    • Primitive data types and objects: C++, Java, C#, D, Delphi, Ada, Go, Swift

    • Everything is an object: Python, Ruby, Javascript, Smalltalk

Primitive Data Types

  • Definition: Data types that are not objects

  • Advantage

    • Efficiency: Primitive data types are faster than objects

    • Simplicity: Primitive data types are easier to implement

  • C

    • base: int, short, long, float, double, char, bool

    • derived: array, pointer, struct, union, enum

All Objects

  • Definition: All data types are objects

  • Advantage

    • Flexibility: All data types can be extended

    • Consistency: All data types have the same syntax

  • Python

    • everything derived from object

    • type is an object too

    • all operations are method calls

    • everything is extensible

Representative Data Types

  • Numeric: int, float, complex, decimal

  • Boolean

  • Character: char, string

  • Enumeration

  • Array, List, Tuple

  • Set, Dictionary

  • Memory address: Pointers and References

  • Composite: struct, union, class

Language Support for Data Types

  • Type checking

    • Static: C, C++, Java, C#, D, Swift

    • Dynamic: Python, Ruby, Javascript

  • Strong typing: type error is always detected

  • Type equivalence

    • Name equivalence: C, C++, Java, C#, D, Swift

    • Structure equivalence: Pascal, Ada, Go

  • Type system

Language Implementation

  • C/C++

    • No run-time support

    • automatic stack

    • manual heap

  • Java

    • JVM: Java Virtual Machine

    • automatic stack and heap

    • garbage collection

  • Python

    • CPython, Jython, IronPython, PyPy implementations behaves differently

    • automatic stack and heap

    • garbage collection

Java Data Types

  • Primitive data types

    • int, float, double, char, boolean

    • Wrapper classes: Integer, Float, Double, Character, Boolean

  • Object data types

    • Automatic heap allocation and garbage collection

    • Stores reference to objects
