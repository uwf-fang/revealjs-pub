C++ Implementation of a List ADT

  • C++ STL standard container std::list, std::vector

  • Make from scratch as templates

  • Example public interface:

    template <typename T>
class List {
 private:
  ///...
 public:
  // allow modification
  T& at(int index);
  // read only
  const T& at(int index) const;
  // index can be size to append to the end
  void insert(int index, const T& value);
  void remove(int index);
  void remove(const T& value);
  void clear();
  int size() const;
  bool empty() const;
  int search(const T& value) const;
  void print() const;
};

Data Structures

  • No data structure is perfect

  • Linked List

    • fast insertion and removal

    • slow indexing

  • Array

    • fast indexing

    • slow insertion and removal

  • Advanced data structures

    • In the cost of complex logics or extra spaces

    • hybrid: linked list with array indices

    • skip list

    • hash table

    • search tree
