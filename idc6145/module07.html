Characteristics and Advantages of Streaming

Characteristics

  • Real-time

  • Continuous

  • Heterogeneous

  • Imperfect

  • Volatile

Advantages of Streaming

  • Real-time insights

  • Scalability

  • Flexibility

  • Continuous monitoring

  • Reduced costs on storage of intermediate data

Drawbacks of Streaming

  • Only implement when the real-time requirement justifies the cost!

  • Data quality issues: no time for data cleaning and validation

  • No backup of historical and intermediate data

  • Added complexity to the pipeline

  • Increased costs

Frameworks and Services

  • Frameworks

    • Apache Flink

  • Services

    • Amazon Kinesis

Checkpoint: Streaming

When to use streaming?

  • Time-sensitive applications - fraud detection, hazard detection

  • Continuous data processing - stock market analysis

Do you really need streaming?

  • It is not free! It is actually expensive!

  • “Real-time” means seconds or less. In many cases, data that is “older” by hours, or days is good enough.

  • E.g. Recommendations based on yesterday’s data is still very useful as human beings do not change every second.
