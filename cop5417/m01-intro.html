Course Description

  • Description:

    This comprehensive course examines fundamental data structures such as arrays, linked lists, trees, graphs, hash tables, and heaps in depth. Students will also learn about important algorithms that are closely related to these data structures, such as sorting, searching, divide-and-conquer, greedy algorithms, etc. Participants must have a solid understanding of programming language fundamentals, including variables, constants, selection statements, iteration, arrays, file input/output operations, and functions, in order to excel in this course. Students will finish the course with a solid foundation in data structures and algorithms, allowing them to confidently tackle complex programming challenges.

  • Keywords:

    Abstract Data Type (ADT); Data Structure; Algorithm; Algorithm Analysis; Sorting; Liked-list; Stack; Queue; Trees; Graphs; Hashing

Course Objective

  • Analyze algorithms in terms of time and/or space complexity.

  • Utilize arrays in computer programs

  • Utilize linked data structures (lists, trees and graphs)

  • Use trees (including height balanced AVL and B-Trees) in a variety of computational problems

  • Describe and implement heaps

  • Describe and construct hash functions and hash tables.

  • Employ algorithms pertaining to graphs to fulfill a variety of computational problems

  • Describe and use simple and complex sorting algorithms

  • Identify characteristics of greedy algorithms and their applicability to computational problems.

  • How to use this textbook

    This is an algorithm book rather than a Java programming book. This course assumes that you have learned the basic concepts of Java programming language before. You can use the first section in the textbook to revisit essential Java concepts used in this course. You will need to learn language details on your own if you are not familiar with Java.
