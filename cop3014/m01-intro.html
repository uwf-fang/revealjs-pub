Course Description

  • Description: An introduction to advanced computational and problem solving techniques. Emphasis on the use of basic programming constructs to create correct, efficient algorithms. Secondary focus on the basic structure and decomposition of programs.

  • Keywords:

    General purpose programming; C++ programming language; Development environment and workflow; Algorithms; Pseudocode; Algorithm analysis;

Course Objective

  • Develop algorithmic solutions to problems, using basic data types, control constructs, and parameter passing.

  • Build program solutions that require simple search and sorting algorithms.

  • Follow a development process where algorithms are developed before implementation begins.

  • Design and implement solutions to problems involving mathematical or scientific principles.

  • Analyze solutions for correctness and efficiency.
