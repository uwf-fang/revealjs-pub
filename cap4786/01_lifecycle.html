Stages

Stages in a data flow pipeline. Downstream stages rely on data generated in upstream stages. These stages are frequently iterative, which means people may go back to early stages whenever changes happens in later stages require adjustment in early stages.

  1. Data acquisition

    • collect data from various sources

  2. Data processing

    • convert raw data to a form that is ready for downstream analysis

  3. Data analysis

    • Not only traditional statistical analysis

    • Various outcomes

      • reports and dashboards

      • visualizations

      • statistical or machine learning/deep learning model

      • optimized procedure/algorithm
