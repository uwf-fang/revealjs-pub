Hadoop

  • History

    • First generation big data technology

    • Inspired by Google’s MapReduce and Google File System paper (2004)

  • Key Characteristics

    • Scalable

    • Fault-tolerant

  • Components

    • HDFS (file system)

    • MapReduce (distributed processing/computing engine)

    • YARN (resource/cluster manager)

Hadoop (Cont)

  • Other characteristics

    • Java-based

    • Modular since 2.0

      • Hadoop ecosystem

    • Commodity hardware

  • Many tools in the ecosystem

    • Pig, Hive, HBase, Sqoop, Flume, Oozie, etc.

Spark

  • The textbook does not discussed it in chapter 1

  • History

    • Emerged as a faster alternative to Hadoop MapReduce

    • Grow to be a general-purpose big data processing engine soon after

  • Key Characteristics

    • In-memory processing

    • scalable

    • Fault-tolerant

  • Components

    • Spark Core

    • Spark SQL

    • Spark Streaming

    • MLlib

    • GraphX

Spark (Cont)

  • Other characteristics

    • Scala-based

    • Supports Java, Python, R, and SQL

    • Run on Hadoop, Mesos, or standalone

Spark vs Hadoop

  • Spark is a monolithic system while Hadoop is now an ecosystem

  • Spark is faster than Hadoop MapReduce

  • Spark is battery-included while Hadoop is more relying on third-party components

  • Spark is way more easier to install and manage than Hadoop ecosystem

Data Serialization

  • Part of the storage aspects in big data lifecycle

  • Serialization is the process of converting an object into a stream of bytes

  • Tools

    • Avro

    • Thrift

    • Protocol Buffers

    • SequenceFile

Columnar Storage

  • Part of data serialization

  • Part of the storage aspects in big data lifecycle

  • Columnar storage is a method of storing data by column rather than by row

  • A more scalable and efficient way to store large data

  • Tools

    • RCFile

    • Parquet

    • ORC

Messaging Systems

  • A powerful tool to join the big data processing pipeline

  • Producer/consumer model

  • Related to data streaming

  • Tools

    • Kafka

    • ZeroMQ

    • RabbitMQ

    • ActiveMQ

    • Flume

NoSQL

  • A general term for new generation of databases that are not relational

  • Use to refer to “Non-SQL” or “Non-relational”

  • Now it is more like “Not Only SQL”

  • A big family of many different types of databases

  • Some Characteristics (not all applies at the same time)

    • Schema-less

    • Distributed

    • SQL-like query language

  • Types

    • Key-value

    • Document

    • Column-family

    • Graph

  • Examples: Cassandra, MongoDB, HBase, CouchDB, Neo4j, etc.

Distributed Query Engine

  • A tool to query data in a distributed environment

  • SQL-like query language

  • Tools

    • Hive

    • Spark SQL

    • Impala

    • Presto

    • Drill

    • Phoenix
