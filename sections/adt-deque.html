Deque Abstract Data Type

  • Overview

  • Behaviors

Overview

  • A deque (double-ended queue) ADT is a sequence/list of elements that can be added to or removed from both ends.

  • Characteristics

    • Homogeneous

    • Add and remove elements at both ends

    • Finite

    • Dynamic size

  • Applications (Mostly out of the scope of this course)

    • Implement a stack or a queue

    • BFS in 2D grid

    • Sliding window maximum/minimum

  • Known as one of the linear ADT because it is usually implemented using a linear data structure. It is not logically linear.

Behaviors

  • enqueue front/back: Insert an element to the front/back of the deque.

  • dequeue front/back: Remove an element from the front/back of the deque.

  • peek front/back: Return the element to be removed from the deque.

  • isEmpty: Return true if the stack is empty, false otherwise.

  • size (optional): Return the number of elements in the stack.
