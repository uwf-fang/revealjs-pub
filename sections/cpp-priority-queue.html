C++ Implementation of Priority Queue ADT

  • Use C++ STL container std::priority_queue

  • Make from scratch as templates

  • Example public interface:

    template <typename T>
class PriorityQueue {
 private:
  ///...
 public:
  void enqueue(const T& value);
  T dequeue();
  T peek() const;
  bool isEmpty() const;
};

Data Structures

  • Heap

  • Sorted array or linked list

    • enqueue: perform a sorted insert (part of the insertion sort algorithm)

    • dequeue: remove the first/last element
