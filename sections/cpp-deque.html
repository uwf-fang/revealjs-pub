C++ Implementation of Deque ADT

  • Use C++ STL container as a deque: std::deque

  • Make from scratch as templates

  • Example public interface:

    template <typename T>
class Deque {
 private:
  ///...
 public:
  void enqueueFront(const T& value);
  void enqueueBack(const T& value);
  T dequeueFront();
  T dequeueBack();
  T peekFront() const;
  T peekBack() const;
  bool isEmpty() const;
};

Data Structures

List of data structures and STL containers that can be used to implement a deque.

  • Partially filled C array

    • better to use a circular array

    • fixed capacity

  • Linked list

    • Ideal

    • doubly, add and remove at the head or tail

  • Other STL containers

    • std::list
