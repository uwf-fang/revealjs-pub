Linked List Example

template <typename T>
class List {  // Doubly-linked list
  Node<T> *head;
  Node<T> *tail;
  int size;
public:
  List();
  ~List();
  void clear();
  void append(const T &data);
  void insertBefore(const T &data, Node<T> *next);
  void remove(Node<T> *node);
  Node<T> *search(const T &data);
  int getSize() const;
};

Choice of Data Structures

  • No data structure is perfect

  • Linked List

    • fast insertion and removal

    • slow indexing

  • Array

    • fast indexing

    • fixed size

    • slow insertion and removal

  • Advanced data structures (FYI)

    • In the cost of complex logics or extra spaces

    • hybrid: linked list with array indices

    • skip list

    • hash table

    • search tree
