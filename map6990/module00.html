Course Description

  • This course provides an introduction to the field of big data in data science.

  • Students will learn about the fundamentals of data mining, data processing, and data analysis, as well as the technologies and tools used to manage large datasets.

  • They will also explore the applications of big data in different industries and gain hands-on experience with popular software platforms and programming languages.

  • By the end of the course, students will be equipped with the knowledge and skills to tackle real-world big data challenges and make informed decisions based on data-driven insights .

Topics

  • Definition and Characteristics of Big Data

  • Big data lifecycle

  • Data storage for big data

  • Data processing for big data

  • Big data analytics

  • Data streaming for big data

  • Cloud computing for big data

  • Data security and privacy for big data

  • Use cases and best practices

Learning Outcomes

  • Understand the fundamentals of big data and the challenges involved in managing and analyzing large datasets

  • Explore the applications of big data in various industries and understand how data-driven insights can inform decision-making

  • Learn about the different technologies and tools used for data mining, data processing, and data analysis

  • Gain hands-on experience with popular software platforms and programming languages used for big data, such as Hadoop, Spark, Python

  • Develop critical thinking skills and the ability to identify patterns, trends, and insights in large datasets

  • Develop a solid foundation in big data concepts and be equipped to tackle real-world data challenges in a variety of settings.
