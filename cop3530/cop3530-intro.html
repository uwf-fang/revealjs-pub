Course Description

  • Description:

    This course covers the design and implementation of algorithms and data structures. Topics include: algorithm analysis, abstract data types, recursion, sorting and searching, lists, stacks, queues, trees, graphs, and hashing.

  • Keywords:

    Abstract Data Type (ADT); Data Structure; Algorithm; Algorithm Analysis; Recursion; Sorting; List; Stack; Queue; Tree; Graphs; Hashing

Course Objective

  • Learn to design and implement algorithms to solve problems

  • Learn C++ fundamentals to construct essential data structures

  • Learn abstraction and modularity to build large programs

  • Learn abstract data types and data structures

  • Learn to analyze the efficiency of algorithms

  • Learn to choose the right data structure for a given problem
