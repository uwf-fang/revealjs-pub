Exponents

  • Definition: An exponent is a quantity representing the power to which a given number or expression is to be raised, usually expressed as a raised symbol beside the number or expression.

  • Example: \(2^3 = 2 \cdot 2 \cdot 2 = 8\)

  • Formula: \(a^n = a \cdot a \cdot a \cdot ... \cdot a\) (n times)

  • Properties

    • Product Rule: \(a^{m+n} = a^m \times a^n\)

    • Quotient Rule: \(a^{m-n} = \frac{a^m}{a^n}\)

    • Power Rule: \((a^m)^n = a^{m \cdot n}\)

    • Negative Exponent Rule: \(a^{-n} = \frac{1}{a^n}\)

    • Zero Exponent Rule: \(a^0 = 1\)

Logarithms

  • Definition: The logarithm of a number is the exponent to which another fixed value, the base, must be raised to produce that number.

  • Example: \(log_2(8) = 3\) because \(2^3 = 8\)

  • Formula: \(log_b(x) = y\) if and only if \(b^y = x\)

  • Properties

    • Logarithmic Identity: \(\log_b(b^x) = x\)

    • Exponential Identity: \(b^{\log_b(x)} = x\)

    • Product Rule: \(\log_b(xy) = \log_b(x) + \log_b(y)\)

    • Quotient Rule: \(\log_b\left(\frac{x}{y}\right) = \log_b(x) - \log_b(y)\)

    • Power Rule: \(\log_b(x^n) = n \cdot \log_b(x)\)

Series

  • Definition: A series in mathematics is the sum of terms in a sequence.

  • Related to the algorithm analysis in computer science

  • Arithmetic series

    • Example: \(1 + 2 + 3 + ... + n = n(n+1)/2\)

    • Formula: \(S_n = \frac{n}{2} \cdot (2a + (n-1)d)\)

    • where:

      • \(n\) is the number of terms,

      • \(a\) is the first term, and

      • \(d\) is the common difference between terms.

  • Geometric series

    • Example: \(1 + 2 + 4 + 8 + ... + 2^n = 2^{n+1} - 1\)

    • Formula: \(S_n = \frac{a \cdot (1 - r^n)}{1 - r}\)

    • where:

      • \(a\) is the first term,

      • \(r\) is the common ratio between terms, and

      • \(n\) is the number of terms

Modular Arithmetic

  • Definition: In mathematics, modular arithmetic is a system of arithmetic for integers, where numbers “wrap around” upon reaching a certain value—the modulus (plural moduli).

  • Example: \(13 \equiv 1 \pmod{6}\) because \(7 = 1 + 6 \cdot 2\)

  • Formula: \(a \equiv b \pmod{n}\) if and only if \(n | (a - b)\)

  • Applications: array indexing, hash functions, pseudorandom number generators, etc.
