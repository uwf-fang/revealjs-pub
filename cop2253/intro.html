Course Description

  • Description:

    Introduction to algorithms and object-oriented programming. Topics include control constructs, looping constructs, parameter passing and arrays. Emphasizes developing fundamental programming skills and software engineering principles in the context of an object-oriented language to solve complex problems in a secure and robust manner.

  • Keywords:

    Object-oriented programming; Java programming language; Algorithms; Control constructs; Looping constructs; Parameter passing; Arrays; Class and object concepts

Course Objective

  • Design and write secure and robust Java programs using existing and newly created classes.

  • Use primitive and object data types.

  • Write methods involving parameter passing, parameter validation, return values, selection and repetition structures.

  • Use single dimension arrays for sorting and searching

  • Perform I/O with GUI and standard methods, including type checking.

  • Model using UML diagrams, design behaviors and attributes of a class and create classes from UML diagrams

  • Design simple class hierarchy involving inheritance and write programs involving inheritance and type casting.
